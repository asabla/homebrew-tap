# Homebrew Tap

Custom Homebrew tap for macOS applications.

## Usage

```bash
brew tap asabla/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `parakatt` | Voice-to-text transcription for the macOS menu bar |

### Install

```bash
brew install --cask asabla/tap/parakatt
```

### Update

```bash
brew upgrade asabla/tap/parakatt
```

### Uninstall

```bash
brew uninstall --cask parakatt
```
